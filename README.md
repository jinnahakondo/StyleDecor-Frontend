# StyleDecor

StyleDecor is a modern appointment and booking management system for a local decoration company.  
It offers both **in-studio consultations** and **on-site decoration services** for homes and ceremonies.

Users can explore decoration packages, check decorator availability, select a date and time, choose a service mode, make secure payments, and track their service status in real time.
